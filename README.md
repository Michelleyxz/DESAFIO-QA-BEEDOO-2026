# DESAFIO-QA-BEEDOO-2026
ste repositório foi criado como parte do desafio técnico para a vaga de Quality Assurance (QA) da Beedoo.  O objetivo deste projeto é demonstrar habilidades de análise de sistema, criação de cenários de teste, execução de testes e registro de bugs, aplicando boas práticas da área de garantia de qualidade de software.
